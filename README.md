# lodash-tpl-loader

A simple [lodash template](https://lodash.com/docs/#template) loader for [webpack](http://webpack.github.io/). 

## Install

`npm install --save-dev lodash-tpl-loader`


## Options

See [lodash template options](https://lodash.com/docs/#template)
